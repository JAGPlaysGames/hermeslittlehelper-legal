Authorization Callback

This page is used as an OAuth redirect target for JAGPlaysGamesHelper account authorization flows when a public HTTPS redirect URL is required.

If you arrived here after signing in, copy the full redirected URL from your browser address bar and paste it only into the secure local setup helper running on your own machine. Do not share authorization codes, client secrets, access tokens, or refresh tokens in public chat.

The app uses authorization only for the creator-approved publishing workflow described on the main website.
