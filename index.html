What this app does

A focused content-operations website for an authorised creator workflow.

X1xJAGx1X Content Studio supports the JAG Plays Games / X1xJAGx1X creator brand by organising approved vertical gaming clips, preparing captions and metadata, showing account connection status, and helping the creator complete platform-specific publishing steps.

The service is not a social network, scraping tool, engagement automation product, or data resale product. It is a creator operations tool for content that the creator owns or is authorised to publish.

All external posting or account changes require explicit creator/operator approval in the intended workflow.

Connect

TikTok Login Kit

The creator connects a TikTok account through TikTok’s OAuth authorization flow. The app uses the returned authorization only for the approved account workflow.

Review

Approval-first operations

Finished clips, titles, captions, hashtags, and upload notes are checked before any publishing action is attempted.

Control

Clear data handling

Privacy, Terms, Support, and Data Deletion pages are visible from this public website and explain how connected account data is handled.

TikTok integration

Limited TikTok access for account authorization.

For the initial TikTok review, this website documents a minimal Login Kit integration. The app should request only scopes that are configured and shown in the review demo.

  • user.info.basic — identifies the authorised TikTok creator account after Login Kit authorization.

Additional TikTok products such as Content Posting API should be requested only when the corresponding upload/publish UI and end-to-end demo video are ready for review.

The app does not use TikTok data for advertising, profiling, resale, unrelated analytics, surveillance, or scraping.
