Support

Last updated: 2026-06-11

JAGPlaysGamesHelper is a private creator operations tool. Support is focused on account connection, TikTok API authorization, approved upload workflows, privacy questions, and deletion requests.

Public creator channels

  • YouTube: @x1xjagx1x
  • X / Twitter: @_X1xJAGx1X_

Support topics

  • Connecting or disconnecting a TikTok account.
  • Understanding requested TikTok scopes.
  • Deleting locally stored app tokens or upload records.
  • Investigating an upload, publish, or API authorization error.
  • Correcting app website, privacy, or terms information.

How to request data deletion

Follow the instructions on the Data Deletion page. You can also revoke TikTok access directly from your TikTok account settings; this prevents further API access using the revoked authorization.

Reviewer note
